Need Help?

Welcome to the ShareKaro Help Center. If you're experiencing any issues or have questions, you're in the right place.

Frequently Asked Questions

  • How do I upload a file?

    On the main page, locate the "File Upload" section. Click the "SELECT FILES" button. A file explorer window will open, allowing you to select one or more files from your device. Once selected, the upload process will begin automatically, and you'll see a progress bar. Upon successful upload, a unique 6-digit access code will be generated for each file, which you can share for downloading.

  • How do I download a file?

    In the "File Download" section on the main page, enter the 6-digit access code provided by the uploader into the input field. Click the "INITIATE DOWNLOAD" button. Your browser will then prompt you to save the file to your device.

  • What types of files are supported?

    ShareKaro supports all file types without any restrictions. You can upload documents, images, videos, archives, and more.

  • How long are files stored?

    For your privacy and to manage storage efficiently, all uploaded files are automatically deleted from our servers after 3 minutes from the time of upload. Please ensure the recipient downloads the file within this timeframe.

  • Is my data safe?

    Yes, your data security is our top priority. All files transferred through ShareKaro are encrypted end-to-end with AES-256 encryption. This means your files are encrypted on your device before being sent and can only be decrypted by the intended recipient, ensuring your data remains private and secure.

  • What's the maximum file size?

    The current maximum file size for uploads is 10 MB. Files exceeding this limit will not be uploaded.

How to Share Text

  • How do I share text?

    Navigate to the "Share Text" section. Type or paste your text into the provided text area. Click the "GENERATE CODE" button. A short access code will be displayed, which you can share with others.

  • How do I retrieve shared text?

    In the "Share Text" section, locate the input field for retrieving text. Enter the access code you received. Click the "RETRIEVE TEXT" button, and the shared text will appear below.

  • How long is shared text stored?

    Shared text snippets are stored for 10 minutes before being automatically deleted to ensure privacy and efficient resource management.

Troubleshooting Tips

  • File upload failed:

    Check your internet connection. Ensure the file size does not exceed 10 MB. If the issue persists, try again or contact support.

  • File not found/expired:

    This usually means the 3-minute storage limit for the file has passed, or the access code was incorrect. Please ask the sender to re-upload the file and provide a new code.

  • Text not found/expired:

    Similar to files, shared text has a 10-minute expiry. Verify the code is correct and ask the sender to re-share if necessary.

  • Page not loading correctly:

    Try clearing your browser's cache and cookies, or try accessing ShareKaro from a different browser.
